    Class GenericSchemaDocument

    JSON schema document.

    A JSON schema document is a compiled version of a JSON schema. It is basically a tree of internal::Schema. This is an immutable class (i.e. its instance cannot be modified after construction).

    ValueT

    Type of JSON value (e.g. Value ), which also determine the encoding.

    Allocator

    Allocator type for allocating memory of this document.
