    Class schedule::Constant

    Constant Schedule.

    The schedule (or schedule segment) has a constant value over the input range. If queried for values outside the input range, the same constant value is returned as well. Configuration Examples:

    • Basic definition:
    "schedule": {
  "type": "constant",
  "value": 42.0
}
    • Short-hand notation:
    "schedule": 42.0
    • With input range:
    "schedule": {
  "type": "constant",
  "value": 42.0,
  "start": 0,
  "stop": 13
}
    • With count:
    "schedule": {
  "type": "constant",
  "value": 42.0,
  "count": 100
}
    Note

    Since the value of this generator is constant, the properties start, stop and count only affect how many times the value is repeated when used as a Segment or generating a discretized set.
