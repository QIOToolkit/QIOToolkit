    Class solver::PopulationAnnealing::CullingEstimator

    The CullingEstimator is used to infer the expected culling ratio for a proposed temperature stepping delta_beta for the "constant_culling" stepping method.

    (It is used by the find_delta_beta method below to iteratively find the appropriate stepping to keep culling approximately constant).
