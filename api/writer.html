    Class Writer

    JSON writer.

    Writer implements the concept Handler. It generates JSON text by events to an output os. User may programmatically calls the functions of a writer to generate JSON text. On the other side, a writer can also be passed to objects that generates events, for example Reader::Parse() and Document::Accept().

    SetMaxDecimalPlaces()

    Sets the maximum number of decimal places for double output.

    This setting truncates the output with specified number of decimal places. For example,

    writer.SetMaxDecimalPlaces(3);
writer.StartArray();
writer.Double(0.12345);                 // "0.123"
writer.Double(0.0001);                  // "0.0"
writer.Double(1.234567890123456e30);    // "1.234567890123456e30" (do not truncate significand for positive exponent)
writer.Double(1.23e-4);                 // "0.0"                  (do truncate significand for negative exponent)
writer.EndArray();

    The default setting does not truncate any decimal places. You can restore to this setting by calling

    writer.SetMaxDecimalPlaces(Writer::kDefaultMaxDecimalPlaces);
