    Class optimal_learning::SimplexIntersectTensorProductDomain

    \rst Domain class for the intersection of the unit simplex with an arbitrary tensor product domain. To that end, this object has a TensorProductDomain object as a data member and uses its functions when possible. See TensorProductDomain for what that means. The unit d-simplex is defined as the set of x_i such that:

    • x_i >= 0 \forall i (i ranging over dimension)
    • \sum_i x_i <= 1

    (Implying that x_i <= 1 \forall i) ASSUMPTION: most of the volume of the tensor product region lies inside the simplex region. \endrst
