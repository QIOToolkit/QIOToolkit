    Class optimal_learning::NormalRNG

    \rst Functor for computing normally distributed (N(0, 1)) random numbers. Uses/maintains an uniform RNG (currently UniformRandomGenerator) and transforms the output to be distributed ~ N(0, 1). .. Note:: seed values take type EngineType::result_type. Do not pass in a wider integer type! .. WARNING:: this class is NOT THREAD-SAFE. You must construct one object per thread (and ensure that the seeds are different for practical computations). \endrst
