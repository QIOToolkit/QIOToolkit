    Class model::ModelRegistry

    Registry for model selection.

    The model registry allows models to be instantiated from their identifier Usage:

    • Registering a new model:
    class MyModel : public [model::BaseModel](xref:classmodel_1_1BaseModel) { public: // Must have a constructor without arguments. MyModel() = default;
// This is the identifier the model is registered with. std::string get_identifier() const override { return "my_model"; } ... } [REGISTER_MODEL(MyModel)](xref:model__registry_8h_1ad6bd7a2bf339ad155cb53db753d98cbc); ```
  * Instantiating a model by identifier:
```c++ auto* my_model = [model::ModelRegistry::create](xref:classmodel_1_1ModelRegistry_1abab837dbb1601ca2bdb9a7d0ea91217d)("my_model"); ```
